The 250kb Club

The WWW has become a bloated mess. Many pages are loading megabytes of Javascript to show you a few kilobytes of content. These things are a cancerous growth on the web and we should stand up against it.

We can make a difference - no matter how small it may seem. The 250kb Club is a collection of web pages that focus on performance, efficiency and accessibility.

If you'd like to suggest a web page to add to this collection, open a ticket on Github or Sourcehut. The site will be reviewed and, if applicable, added to the list below.

If your pages exceeds 250kb, you might consider 1MB.club which is the inspiration for this page.
